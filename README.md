# PCF Energy Efficiency Rating Control

## Overview

Energy Efficiency Rating Control allows you to set rating via slider and it will show the visual representation of the rating below.

## Download

[![download](https://user-images.githubusercontent.com/14048382/27844360-c7ea9670-6174-11e7-8658-80d356c1ba8f.png)](https://github.com/DynamicsNinja/PCF-Energy-Efficiency-Rating-Control/releases/latest)

## Configuration

| Field  | Description                                                  |
| ------ | ------------------------------------------------------------ |
| Rating | Whole number field that will be populated after you set the value by moving the slider. |
